#Movie Bot
Хороший бот для поиска фильмов и сериалов.

###Основные фичи:
- Поиск по названию
- Поиск по категориям
- Еще какие-то фичи

###Установка
1. Склонируйте репозиторий
2. Выполните `pip install -r requirements.txt`
3. Добавьте в корень приложения файл `.env` с вашими токенами
4. Запустите бота `python main.py`
